Los atributos mas comunes de las etiquetas inputs son

  • name: identifica un campo de entrada en particular
  • value: define el valor predeterminado de un campo de entrada o de seleccion
  • placeholder: muestra un texto de sugerencia en el campo de entrada cuando esta vacio
  • required: indica que un campo de entrada es obligatorio y debe completarse
  • disabled: desactiva el campo de entrada, el usuario no puede interactuar con este
  • readonly: hace que el campo de entrada sea solamente de lectura
  • autocompletepermite que el navegador completar automaticamente los campos de entrada con valores almacedados previamene por el usuario, esto puede ser con "on" y "off"
  • min y max: para definir un maximo y un minimo en aquellos inputs que tienen que ver con valores numericos
  • pattern: permite especificar una expresion regular que el valor ingresado debe coincidir para ser valido
  • oninvalid: para personalizar el mensaje de error usando js con "this.setCustomValidity('Mensaje')"
  • oninput: para personalizar el mensaje de bueno usando js con "this.setCustomValidity('Mensaje')"
  • checked: La casilla esta marcada como determinada en el formulario hasta cambiarlo
  • multiple: agregar varios emails separados por comas
  • selected: objeto de un checkbox o select seleccionado de forma predeterminada
  • multiple: permite que el checkbox o selecte pueda ser de seleccion multiple por el comando ctrl + click
