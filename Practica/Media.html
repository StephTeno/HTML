Imagenes con img

Las imagenes se pueden mostrar bajo la etiqueta de una sola apertura llamada img, estas imagenes pueden ser propias o sacadas del internet pero... Â¿Como puedo hacerlo? simple, con el atributo propio src que significa source

Sin embargo, como buena practica se utiliza el atributo alt ya que proporciona un texto alternativo para el usuario si por alguna razon no pueda verla, es obligatorio colocarlo

El atributo usemap tiene un uso similar al de las anclas pero en este caso para un mapa de imagenes

Audios con 'audio' & Videos con 'video'

Los audios y videos se pueden "invocar" bajo la etiqueta de apertura y cierre llamada audio y video respectivamente, estos tienen la gracia que solamente se coloca de forma interna, es decir que hay que tenerlas agregadas dentro de los archivos

Este elemento posee varios atributos como ejemplo

  • autoplay: Especifica que el audio/video se reproducira en cuanto se cargue la pagina, esta no tiene valor

  • controls: Especifica que los controles del audio/video se deben mostrar, esta no tiene valor

  • loop: Especifica que el audio/video se reproducira en bucle ininterrumpidamente, esta no posee valor

  • muted: Especifica que la salida de audio/video sebe ser silenciada, solamente hasta que se desmutee con los controles, esta no posee valor

  • preload: Especifica si al cargarse la pagina debe empezar a reproducirse automaticamente, esta si tiene que poseer un valor tales como: autoplay, metadata, none

  • src: Especifica la direccion URL o local donde se encuentra el archivo de audio/video, el valor es la URL

Iframe

Un iframe en HTML se utiliza para mostrar una pagina web dentro de otra pagina web simplemente con la etiqueta 'iframe' que poseen su atributo principal el src que es el url donde se va hacer el mini pantallita dentro de la pagina

Un ejemplo sacado de YouTube el trailer del 3er juego de FNAF

Y se puede utilizar como marco de destino para un enlace, usando el target del enlace que debe hacer referencia al name del iframe
