Etiquetas de listas

Los elementos se enumeran con la etiqueta "li" que es contenida por las etiquetas "ol" y "ul"

Y estas pueden tener como atributos los globales como el "accesskey", "tabindex", "title" y tambien tiene uno propio como es el "value" esto hace que los elementos de la lista tengan un valor

Listas ordenadas "ol"

Tiene el atributo "type" que especifica el tipo de enumeracion, colocando "1" para numeros, "a" para letras minusculas, "A" para letras mayusculas, "i" para numeros romanos minusculos, "I" para numeros romanos mayusculos

Tambien tiene el atributo "start" que especifica el numero con el cual se comenzara la enumeracion

Esta el reversed que si se establece, la lista de elementos para a ser inversa

Listas desordenadas "ul"

Tiene el atributo "type" que especifica el tipo de simbolo que tendra la enumeracion, colocando "disc" que es un circulo relleno, "circle" el contorno de un circulo y "square" que es un cuadrado

Tablas

Toda la tabla se contiene en una etiqueta llamada "table", cuyas filas se etiquetan "tr"

Esta tambien puede ser titulada con la ayuda de la etiqueta "caption" que se agrega antes de "thead" pero despues de la etiqueta contenedora "table", tiene como unico atributo el "align" que pueden ser "top" que es la parte superior, "bottom" es la parte interior

Se puede separar semanticamente en encabezado, cuerpo y pie de tabla

El encabezado tiene de etiqueta "thead", que contiene una "tr" como unica fila y "th" segun la cantidad de columnas que se desean

La etiqueta "colgroup" se utiliza para agrupar varias columnas de una tabla para asi establecer estilos y comportamienos similares, se coloca antes del "thead" o "tbody", tiene como atributos "span" que especifica el numero de columnas que se incluyen en el grupo

Definiciones

La etiqueta "dl" es la contenedora para crear una lista de definicion, que se compone de terminos con etiqueta "dt" y sus definiciones correspondientes con la etiqueta "dd" siendo que despues de una "dt", viene su "dd"

Tiene como atributo el "compact" que se usa para especificar que la lista debe de contraerse

Las definiciones pueden ser etiquetas contenedoras!!!!, se le puede agregar "p", "a", etc.

Los terminos tambien pueden ser imagenes
