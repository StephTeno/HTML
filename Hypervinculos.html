Atributos de la etiqueta "a"

Esta puede tener tanto las propias como las globales

  1. href: Esta indica a donde se va dirigir al usuario cuando se de clic
  2. target: Define como se abrra el recurso enlazado, _self abre el recurso en la misma ventana o pestaña, _parent abre el recurso en el marco padre, si lo hay, _blank abre el recurso en una nueva ventana o pestaña y _top abre el recurso en la ventana principal del navegador
  3. download: Indica que el recurso enlazado debe descargarse en lugar de ser mostrado
  4. rel: Define la relacion entre la pagina actual y el recurso vinculado. Por ejemplo el nofollow indica que el enlace no debe de ser seguido por los motores de busqueda, otro valor comun es noopener noreferrer que mejora la seguridad del sitio web siendo el noopener indica que la ventana abierta no debe de tener acceso a la ventana que se abrio el enlace, el noreferrer indic que el navegador no debe enviar la indormacion de referencia del sitio web de la pagina actual al vinculado
  5. type: Indica el tipo de archivo al que se enlaza. Este atributo se usa comunmente para enlazar archivos de imagen, video o audio
